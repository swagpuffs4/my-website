GitHub Pages upload package for Erika Guardamondo's portfolio

Files included:
- index.html
- .nojekyll

How to publish on GitHub Pages:
1. Create a new GitHub repository.
2. Upload the files from this folder to the root of the repository.
3. Commit the files.
4. In GitHub, open the repository.
5. Go to Settings -> Pages.
6. Under Build and deployment, set Source to Deploy from a branch.
7. Choose Branch: main and Folder: /(root).
8. Click Save.
9. Wait a few minutes for GitHub Pages to publish the site.
10. Open the URL GitHub shows in the Pages section.

Notes:
- This site is a static HTML file.
- The resume and portfolio downloads are embedded into the HTML.
- .nojekyll is included to avoid unnecessary Jekyll processing.
